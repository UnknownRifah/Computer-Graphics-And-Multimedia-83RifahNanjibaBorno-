# OpenGL Two Cyan Obtuse Triangles (GLFW)

## Description  
This project is created as part of a course assignment.  
It opens an OpenGL window with an orange background and draws two cyan-colored obtuse triangles at two distant corners of the screen using C++ and GLFW.  
The window title shows the author's full name.  
Pressing the first letter of the author's name closes the window.  

## Features  
- Orange background window  
- Two cyan colored obtuse triangles  
- Window title displays full name  
- Keyboard interaction implemented  
- Pressing **R** closes the window  
- Program runs on local computer (verified via terminal output)  

## Technologies Used  
- C++  
- OpenGL  
- GLFW  
- GLAD  
- g++  

## How to Run  
1. Open the project folder in VS Code  
2. Open terminal inside the folder  
3. Compile the project using:  
   make
4. Run the executable:
   ./build/main.exe
5. Press **R** key to close the window
